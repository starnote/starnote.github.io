## 简介
星空笔记，基于Hive的区块链笔记。它是一个去中心的应用，数据存储在Hive区块链上。 它和比特币的离线钱包类似，可以很方便地在本地构建，无需服务器。

当然，为了方便大家使用，我把它上传到github page中，大家可以直接访问：https://starnote.github.io
